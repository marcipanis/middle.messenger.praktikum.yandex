# chat-project #

------
Разработка веб-приложения «Чат» в рамках обучения на курсе "Мидл фронтенд-разработчик"
от Яндекс.Практикума.


### Команды запуска: ###
+ старт сервера в режиме разработки

      npm run dev
+ сборка проекта

      npm run build
+ сборка проекта  и старт сервера для  раздачи статики на порту 3000

      npm run start
